# Personal Data Collector

A beginner-friendly Python project that demonstrates how to collect user input, work with different data types, display memory addresses, and perform simple calculations.

## 📌 Project Description

This program collects basic personal information from the user, including:

* Name
* Age
* Height (in meters)
* Favorite Number

The program then displays:

* The entered values
* Their respective data types
* Their memory addresses using the `id()` function
* An approximate birth year based on the user's age

This project is ideal for beginners learning Python fundamentals such as variables, data types, user input, type conversion, and output formatting.

---

## 🚀 Features

* Takes user input using `input()`
* Converts input into appropriate data types (`int`, `float`)
* Displays variable data types using `type()`
* Shows memory addresses using `id()`
* Calculates approximate birth year
* Uses formatted and organized console output
