# Pattern Generator and Number Analyzer (Python CLI Project)

## Overview
This is a simple Python command-line program that:
- Generates different patterns
- Analyzes a range of numbers (even/odd + sum)
- Runs in a loop until user exits

---

## Features

### 1. Pattern Generator
Supports 4 patterns:
- Number Pattern
- Pyramid Pattern (^)
- Dollar Pattern ($)
- Star Pattern (*)

### 2. Number Analyzer
- Takes a start and end range
- Prints whether each number is even or odd
- Calculates total sum of the range

### 3. Exit Option
- Safely stops the program

python main.py
