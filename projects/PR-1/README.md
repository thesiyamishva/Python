# Mishva Print Program

## Description

This is a simple Python program that prints the word **"mishva"** five times on the screen.

## Code

```python
print("mishva")
print("mishva")
print("mishva")
print("mishva")
print("mishva")
```

## Output

```
mishva
mishva
mishva
mishva
mishva
```

## Requirements

* Python 3.x

## How to Run

1. Save the code in a file named `main.py`.
2. Open a terminal in the project folder.
3. Run:

```bash
python main.py
```
